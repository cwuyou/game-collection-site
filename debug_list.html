How to Get Embed Code

  1. Select the Game: Choose the game you want to embed. Navigate to the game's page.
  2. Browse the Game Page: On the game page, scroll down to find the embed code. Look for a section labeled "Embed URL".
  3. Copy the Embed URL: Copy the embed URL provided on the page. This URL is essential for embedding the game on your website.

How to Add Embeddable Games to Your Website

  1. Retrieve Embed Code: Proceed to the game page and obtain the embed URL as explained above.
  2. Open Your Website's HTML File: Access your website's HTML file using a code editor or content management system.
  3. Insert Embed Code: In the HTML file, insert the embed code within an <iframe> structure. Replace the src attribute with the previously copied embed URL.
  4. Adjust Parameters: Modify parameters such as width, height, frameborder, or scrolling to suit your website's design.
  5. Save and Test: Save the HTML file and preview it on a web browser. Test the embedded game to ensure it is functioning correctly.

By following these steps, you can seamlessly integrate embeddable games into your website.

Example: Below is an example of an iframe code of Highway Traffic Game with adjusted parameters:

 <iframe width="1060" height="650" src="https://www.onlinegames.io/games/2022/unity/highway-traffic/index.html" frameborder="0" scrolling="no" ></iframe>
